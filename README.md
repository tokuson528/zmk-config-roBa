## レイヤー構成

| レイヤー | 名前 | 呼び出し方 | 役割 |
|---|---|---|---|
| 0 | Default | 常時 | 通常のキー配列。英字・記号・修飾キー |
| 1 | ShortCut | 変換を長押し | よく使うショートカット |
| 2 | Num | SPACEまたはBackspaceを長押し | 数字・計算記号・括弧などの記号 |
| 3 | Arrow | ENTERまたは無変換を長押し | 矢印・Home/End・タブ切り替え |
| 4 | Mouse | トラックボール操作で自動 | マウスクリック（左・中・右） |
| 5 | Scroll | Iを長押し | トラックボールでスクロール |
| 6 | BT/Boot | 変換を長押し | Bluetooth接続切り替え・ブートローダー |

## コンボ

| キー | 動作 |
|---|---|
| S + D | Tab |
| D + F | Shift + Tab |
| A + S | 無変換 |
| C + V | = |
| L + ' | " |
| F + Space | 英語切り替え |
| J + Enter | 日本語切り替え |

## 長押しショートカット（Defaultレイヤー）

| キー | タップ | 長押し |
|---|---|---|
| A | A | Cmd+A（全選択） |
| X | X | Cmd+X（切り取り） |
| S | S | Cmd+S（保存） |
| C | C | Cmd+C（コピー） |
| V | V | Cmd+V（貼り付け） |
| G | G | Cmd+G（次を検索） |
| R | R | Cmd+R（更新） |
| Backspace | Backspace | NUMレイヤー |
